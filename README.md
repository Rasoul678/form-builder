## description

This is a form builder. It allows you to create forms from json and save them to a database or any of your choice.

## installation

1. Clone the repo

```bash
git clone git@github.com:Rasoul678/form-builder.git
```

2. Install dependencies

```bash
npm install
```

or

```bash
yarn
```

3. Run the app

```bash
npm run dev
```
